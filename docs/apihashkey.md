API Management
1. Sub-accounts are a type of account that allows you to create multiple accounts to practice different trading strategies and mitigate risks. HashKey Exchange provides powerful APIs that allow you to access market data and perform automated trading.

2. Each account can create up to 5 sets of API Keys.

3. Please do not disclose your API Key to prevent asset loss. For security reasons, API Keys must be bound to IP addresses, with a maximum of 10 IP addresses per API Key. Enter a single IP address directly, and separate multiple IP addresses with a comma, for example: 192.168.1.1,192.168.1.2,192.168.0.1,192.168.0.1 .
